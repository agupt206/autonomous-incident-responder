== RUNBOOK: Payment Gateway Connection Refused ==
ERROR CODE: ECONNREFUSED at /payment-gateway

ROOT CAUSE 1: The 'payment-service' pod is restarting.
CHECK: kubectl get pods -n payment
FIX: Wait 60 seconds. If it loops, check logs for OOMKilled.

ROOT CAUSE 2: The legacy firewall dropped the packet.
CHECK: curl -v telnet://10.0.0.5:8080
FIX: If timeout, run: 'sudo iptables -F' on the gateway node.

ROOT CAUSE 3: API Key Expired.
CHECK: grep "401" /var/log/payment.log
FIX: Rotate key in Vault.
